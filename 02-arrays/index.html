Introduction to arrays

Arrays are a data structure that can store a collection of elements. The elements can be of the same type or different types. In JavaScript, arrays are zero-indexed, which means that the first element is at index 0, the second element is at index 1, and so on.

Let's say that we have a certain amount of physicial books in our house and we need to set some shelves up so we can collocate our books.

We need to think of a few things to get this done:

  • How many books do we have?
  • How many shelves do we need?
  • How many books can each shelf hold?

Let's say we have 10 books, we need 3 shelves, and each shelf can hold 4 books. In this case, each shelve will represent an array and the books will be the elements of the array.

In JavaScript, an array is created by using square brackets [].

Arrays are indexed starting at 0. So, the first element of an array is at index 0, the second element is at index 1, and so on.

Here is how we can represent the example above using JavaScript:

        
const books = ['book1', 'book2', 'book3', 'book4', 'book5', 'book6', 'book7', 'book8', 'book9', 'book10'];

const shelves = [
    ['book1', 'book2', 'book3', 'book4'],
    ['book5', 'book6', 'book7', 'book8'],
    ['book9', 'book10']
];

Now let's construct a class that will be used to interact with the books and shelves:

        
class BookShelves {
    constructor(books) {
        this.books = books;
    }

    accessBook(index) {
        return this.books[index];
    }

    addBookAtIndex(index, book) {
        this.books[index] = book;
    }
}

In class above, we wrote the following methods:

  • constructor: This is a special method that is called when an object is created from the class. It initializes the object with the given books.
  • accessBook: This method takes an index as an argument and returns the book at that index in the array.
  • addBookAtIndex: This method takes an index and a book as arguments and adds the book to the array at the given index.

Some programming languages support array capacity, meaning that you can define the maximum number of elements that an array can hold. In others, like JavaScript, arrays do not have a fixed capacity. You can add or remove elements from an array at any time.
